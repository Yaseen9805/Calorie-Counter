# 🔥 Calorie Counter

A dynamic calorie tracking application built with **HTML, CSS, and JavaScript**. It helps users manage their daily calorie intake by tracking meals and exercise.

---

## 🚀 Features

* Set daily calorie budget
* Add multiple entries for meals and exercise
* Calculate consumed, burned, and remaining calories
* Input validation and error handling
* Clear form functionality

---

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation, Form Handling)

---

## ⚙️ How It Works

* Enter your daily calorie budget
* Add food and exercise entries dynamically
* Submit to calculate remaining calories
* Displays surplus or deficit with summary

---

